## About

RabbitMQ Docker image, but with some additional configuration:

**Plugins:**  

- [Delayed Message Exchange](https://github.com/rabbitmq/rabbitmq-delayed-message-exchange)

## Install

```sh
$ docker pull sammlerio/rabbitmq
```
